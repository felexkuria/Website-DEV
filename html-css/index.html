HTML & CSS Basis Summary

HTML Summary

HTML (HyperText Markup Language) is the heart of every webpage. It defines the structure of the webpage and annotates the content to tell the browser what to display and provide extra meaning. Without HTML, text would always be "just text" - there would be no semantic difference between titles, normal text, subtitles etc. You also wouldn't be able to add images or links. Learn more about available HTML elements on the MDN HTML element reference.

  • DOCTYPE declaration must be the first line of HTML document to specify HTML version
  • The <html> element is the root element containing all other elements
  • The <head> section contains metadata like title, character encoding, and CSS links
  • The <body> section contains all visible content that appears on the webpage
  • Elements must be properly nested and closed to maintain valid HTML structure

CSS Summary

CSS (Cascading Style Sheets) is a language that allows you to define style definitions for your HTML document. These definitions will be picked up by the browser and control how the content will be displayed (e.g. that some text is red, which font size it should have etc.)

  • Specificity determines which CSS rule applies when multiple rules target the same element
  • Inheritance allows child elements to inherit certain CSS properties from their parent elements
  • Pseudo-selectors like :hover and :first-child allow styling elements based on state or position
  • Class selectors (.classname) allow targeting specific elements with matching class attributes
